Datatypes

Double quotes (" ") or single quotes (' ') can be used to specify strings.
Triple double quotes or single quotes can be used to make comments.
A new line is use to end a line of code
The forward slash is used to go to the next line prematurely
Names are case sensitive and cannot start with numbers but they contain letters, underscores and numbers.

Datatypes can be mutable or immutable. Mutable datatypes are those that when we change data we do it in place, we do not copy them in a new memory addresseach at a time i.e if we type y=x and modify y, x is also affected. Immutable datatypes do not change. Examples of mutable datatypes are lists, dictionaries and user defined types, and immutable are strings, integers, float and tuples.

  1. Tuples are defined using parenthesses and commas ie
    tu=(23, 'abc', 'def', 4.4)
  2. Lists are defined using square brackets and commas ie
    li=['abc', 23, 4.5]
  3. Dictionaries are defined using curly brackets ie
    d={'user':'hellen', 'pswd':1234}

GUI programming with Tkinter

Nearly every GUI program will be written contain the following lines:
tkinter import *
root=Tk()
mainloop()
The first line imports all GUI stuff from Tkinter module. The second line will create a window on the screen which is called a root. The third line puts the program into what is essentially a long- running loop
